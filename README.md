
# Node.js Folder Clutter Cleaner

A simple Node.js utility that automatically organizes messy folders by moving files into categorized folders based on their file extensions.

This project is perfect for beginners learning Node.js File System (fs) and backend fundamentals.

## Features

- Automatically organizes files by type
- Creates folders if they don’t exist
- Skips existing folders safely
- Uses only core Node.js modules
- Fast and easy to run



## Before

    ├── image.png
    ├── resume.pdf
    ├── video.mp4
